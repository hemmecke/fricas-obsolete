HankelPackage R¶

Package to generate Hankel matrices.

HankelMatrix: List R -> Matrix R

HankelMatrix([a_0, a_1, a_2, ..., a_{2n}]) builds the Hankel matrix [ a0, a1, …, an ] [ a1, a2, … ] [ … ] [ an, …, a2n]
